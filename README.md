# ALQAS AL IRAQI - Landing Page

This is a simple static website (HTML/CSS/JS) designed for GitHub Pages.

## Edit these in `script.js`
- `ORDER_LINKS` (Talabat / Snoonu / Keeta)
- `MAPS_LINK` (Google Maps pin)
- `SOCIAL_LINKS` (Instagram / TikTok / Snapchat)

## Behavior
- Every generic "Order Now" button opens a popup with options:
  Talabat / Snoonu / Keeta / WhatsApp / Call

## GitHub Pages
Settings → Pages → Deploy from a branch → main → /(root)
